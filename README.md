📖 Penjelasan
Zuan Kernel Manager (ZKM) adalah alat manajemen kernel Android yang canggih, dibangun kembali dari basis proyek Rve Kernel Manager yang legendaris.
Proyek ini lahir dari keinginan untuk menyempurnakan pengalaman pengguna. Kami telah melakukan rewrite dan modifikasi ekstensif untuk menghadirkan antarmuka pengguna (UI) yang jauh lebih modern, tema yang memanjakan mata, serta perbaikan stabilitas di berbagai sektor. ZKM bukan sekadar rebranding, melainkan evolusi dengan fitur yang lebih kaya dan optimalisasi kode untuk performa yang lebih responsif.

✨ Fitur Utama
ZKM membawa semua fitur handal dari Rve dengan tambahan sentuhan modern:
🎨 Tampilan & UI
Desain Material Baru: Antarmuka yang didesain ulang sepenuhnya untuk navigasi yang lebih intuitif.
Tema Indah: Pilihan tema baru yang lebih elegan dan modern (Dark Mode & Light Mode yang disempurnakan).
Animasi Halus: Transisi antar menu yang lebih fluid.
⚡ Performa & Kontrol
Dashboard Komprehensif: Monitor suhu, frekuensi CPU, dan penggunaan memori secara real-time.
CPU & GPU Tuning: Atur Governor, Frekuensi Min/Max, dan Boost dengan mudah.
Memory Management: Low Memory Killer (LMK), Virtual Memory, dan pengaturan ZRAM.
Battery Controls: Pengaturan wakelocks dan charging control (jika didukung kernel).
🛠 Utilitas Tambahan
Build.prop Editor: Edit properti sistem dengan aman.
Flasher: Flash file zip atau image kernel langsung dari aplikasi.
Script Manager: Jalankan skrip shell kustom saat boot.
Perbaikan Bug: Mengatasi berbagai force close dan glitch yang ada pada versi basis sebelumnya.
📥 Instalasi
Pastikan perangkat Android kamu sudah memiliki akses ROOT (Magisk/KernelSU).
Unduh ZKM-v1.0.apk dari halaman Releases.
Install APK seperti biasa.
Berikan izin Root saat diminta.
Nikmati kontrol penuh atas kernel kamu!
