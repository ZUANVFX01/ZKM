## üìñ Description

**Zuan Kernel Manager (ZKM)** is an advanced Android kernel management tool, rebuilt from the legendary Rve Kernel Manager project base.

This project was born from the desire to perfect the user experience. We have performed extensive rewrites and modifications to deliver a much more modern user interface (UI), eye-pleasing themes, and stability improvements across various sectors. ZKM is not just a rebranding, but an evolution with richer features and code optimization for more responsive performance.

## ‚ú® Key Features

ZKM brings all the reliable features from Rve with additional modern touches:

### üé® Appearance & UI
- **New Material Design**: Completely redesigned interface for more intuitive navigation
- **Beautiful Themes**: New, more elegant and modern theme choices (Enhanced Dark Mode & Light Mode)
- **Smooth Animations**: More fluid transitions between menus

### ‚ö° Performance & Control
- **Comprehensive Dashboard**: Monitor temperature, CPU frequency, and memory usage in real-time
- **CPU & GPU Tuning**: Easily adjust Governor, Min/Max Frequency, and Boost settings
- **Memory Management**: Low Memory Killer (LMK), Virtual Memory, and ZRAM settings
- **Battery Controls**: Wakelock settings and charging control (if supported by kernel)

### üõ† Additional Utilities
- **Build.prop Editor**: Safely edit system properties
- **Flasher**: Flash zip files or kernel images directly from the app
- **Script Manager**: Execute custom shell scripts at boot
- **Bug Fixes**: Resolved various force closes and glitches present in the base version

## üì• Installation

1. Ensure your Android device has ROOT access (Magisk/KernelSU)
2. Download ZKM-v1.0.apk from the Releases page
3. Install the APK as usual
4. Grant Root permission when prompted
5. Enjoy full control over your kernel!
