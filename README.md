## üìñ Description

**Zuan Kernel Manager (ZKM)** is an advanced Android kernel management tool, rebuilt from the **Rve Kernel Manager** project base.

This project focuses on perfecting the user experience. We have performed extensive rewrites and modifications to deliver a modern **User Interface (UI)**, refined themes, and stability improvements across various sectors. ZKM is an evolution designed for better responsiveness and richer features.

## ‚ú® Key Features

ZKM enhances the original base with modern touches and new utilities:

### Appearance & UI
* **Material Design** - Redesigned interface for intuitive navigation.
* **Themes** - Enhanced Dark Mode & Light Mode support.
* **Animations** - Fluid transitions and optimized layout.

### Performance & Control
* **Dashboard** - Real-time monitoring for Temp, CPU Freq, & RAM.
* **CPU/GPU Tuning** - Adjust Governor, Min/Max Freq, and Boost.
* **Memory Management** - LMK, Virtual Memory, and ZRAM settings.
* **Battery** - Wakelock blocker & Charging control support.

### Utilities
* **Build.prop Editor** - Safely edit system properties.
* **Flasher** - Flash zip/img directly (Powered by Horizon logic).
* **Script Manager** - Execute custom shell scripts at boot.

## üì• Installation

1.  Ensure your Android device has **ROOT access** (Magisk/KernelSU/APatch).
2.  Download `ZKM-v1.0.apk` from the **[Releases Page](../../releases)**.
3.  Install the APK.
4.  Grant Root permission when prompted.
